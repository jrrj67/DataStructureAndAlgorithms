# 🚀 Guia de Estudos — Data Structures & Algorithms (DSA) em C#

Este guia reúne exercícios essenciais para evoluir em algoritmos e estruturas de dados — do básico ao nível avançado.

---

## 🧠 1. Arrays e Strings
> Base de tudo.

### 🔹 Exercícios
- 🔄 Inverter um array sem usar `Array.Reverse`
- 🔝 Encontrar o maior e o menor valor
- ♻️ Remover duplicados de um array ordenado
- 🔁 Rotacionar array  
  _Ex: `[1,2,3,4] → [3,4,1,2]`_
- 🔤 Verificar se duas strings são anagramas
- 🔍 Encontrar o primeiro caractere não repetido
- 🔎 Implementar `strStr()` (buscar substring)
- 📦 Compressão de string  
  _Ex: `aaabb → a3b2`_

## 💡 Dica prática (ESSENCIAL)

Se você quer evoluir rápido em C#:

- 🛠️ **Implemente tudo do zero**  
  _(sem `List<>`, `Dictionary<>` no início)_
- 🔄 Depois refaça usando libs do .NET
- 🎯 Foque em:
  - Complexidade → `O(n)`, `O(log n)`
  - Uso de memória
  - Código limpo e legível
